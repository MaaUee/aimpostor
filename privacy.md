Privacy Policy for AImpostor

Last updated: December 2025

AImpostor is a pass-and-play party game created by Manuel Serra.
We respect your privacy and are committed to protecting your personal information.

This Privacy Policy explains what data the app collects, how it is used, and what rights you have.

⸻

1. Information We Do Not Collect

AImpostor does not collect, store, or transmit any personal information such as:
	•	Names
	•	Email addresses
	•	Account information
	•	Location data
	•	Contacts
	•	Device identifiers for tracking

The app does not require an account and does not track users.

⸻

2. Game Data Stored Locally

All in-game data—including player names, roles, topic lists, and match progress—is stored locally on the user’s device.
This local data is never sent to any server and is removed when the game ends or the app is closed.

⸻

3. AI-Generated Topic Lists

When users request an AI-generated topic list, the following data is sent to our backend:
	•	The text the user enters (the theme word)
	•	The selected app language (“de”, “en”, or “it”)

No personal data is included in these requests.

This information is used solely to generate the requested list and is not stored beyond standard server logs.

⸻

4. Firebase Backend

AImpostor uses Google Firebase Cloud Functions to run the backend that generates AI topics.
Firebase may automatically log basic technical data such as:
	•	IP address
	•	Device or browser information (from standard headers)
	•	Timestamps
	•	Request metadata
	•	Error and crash logs

This data is collected automatically by Firebase for security, performance monitoring, and debugging.
We do not use this data to identify users.
We do not store or export this data outside Firebase’s built-in logging systems.

Firebase Privacy Policy:
https://firebase.google.com/support/privacy

⸻

5. OpenAI API

Our Firebase backend sends the user-provided theme word to the OpenAI API to generate a topic list.
OpenAI may store request data temporarily for operational and abuse-prevention purposes.

No personal information is ever included in these requests.

OpenAI Privacy Policy:
https://openai.com/policies/privacy-policy

⸻

6. Advertising (AdMob)

AImpostor uses Google AdMob Rewarded Ads.
AdMob may collect anonymized data such as:
	•	General device information
	•	Approximate location
	•	Advertising interactions
	•	Identifiers used for ads (Google Ad ID / IDFA)

This data is processed according to Google’s policies and is not linked to personal identifying information by us.

Google Privacy Policy:
https://policies.google.com/privacy

Users may manage ad personalization settings through their device.

⸻

7. Children’s Privacy

AImpostor is rated 4+.
The app does not knowingly collect personal data from children.

If you believe a child has provided personal information, please contact us and we will take appropriate action.

⸻

8. Changes to This Privacy Policy

We may update this Privacy Policy as needed.
Changes will be posted on this page with an updated date.
